# NoHablas
This addon blocks LATAM realms from your Premade Group Finder experience. This is done automatically and requires zero configuration or commands. If you wish to toggle functionality of this addon, please Enable/Disable it in the addons menu.

## Why use NoHablas?
Some NA realms primarily operate in languages other than English, which can cause coordination issues in Premade Group Finder content. NoHablas filters listings and applicants by specific realm names to improve communication and group consistency.

This addon does not seek to discriminate against players or communities. It is a gameplay-focused filtering tool intended to reduce language-related friction in organized group content.

## What NoHablas does
1. Groups where the leader is from a LATAM Realm will be **hidden** from the listed results.
2. If a LATAM Realm player applies to one of your groups, they will be **automatically declined**.
